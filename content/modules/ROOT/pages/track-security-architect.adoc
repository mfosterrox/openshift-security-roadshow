== Role overview

Define platform-wide security strategy: governance, multi-cluster posture, identity architecture, zero-trust patterns, and exception lifecycle.

Core responsibilities (framework):

* Secure platform architecture patterns
* Governance and zero-trust policy definition
* Automation strategy for vulnerability and assurance workflows
* Mentoring and maturity uplift
* Admission / policy strategy
* Security infrastructure portfolio (identity, logging, runtime, policy)
* Align security with product and delivery goals
* Multi-cluster policy propagation
* Govern control exceptions
* Workload identity and trust issuance design
