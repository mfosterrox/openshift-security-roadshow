== Role overview

Embed security in delivery workflows: policy gates, supply-chain controls, and feedback loops from runtime back into build and release.

Core responsibilities (framework):

* Secure CI/CD and deployment pipelines
* Secure infrastructure-as-code and build artifacts
* Admission and related policy enforcement
* Threat modeling outcomes into monitoring and IR playbooks
* Workload isolation and zero-trust patterns in delivery
* Artifact signing and attestation verification gates
* Runtime detection tuning and signal quality
* Feed production learnings back into build/test/release
