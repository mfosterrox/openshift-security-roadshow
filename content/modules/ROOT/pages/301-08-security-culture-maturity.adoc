**Duration:** ~20 minutes

== Overview
Lead organizational security culture transformation by developing enablement programs, establishing documentation standards, creating internal security guilds, and building maturity roadmaps that guide teams toward security excellence.

Stand up a practical security enablement approach—docs, coaching, and a maturity roadmap—so teams improve practices without security becoming theater.

== Why it matters

Security culture is foundational to effective security:

* **Security by Design**: Teams build security into their work from the start
* **Shared Responsibility**: Security becomes everyone's responsibility
* **Continuous Improvement**: Teams continuously mature their security practices
* **Knowledge Sharing**: Best practices spread across the organization
* **Reduced Friction**: Security becomes an enabler, not a blocker
* **Sustainable Security**: Security practices become self-sustaining

== What does it solve

Organizations face security culture challenges:

* Security is seen as a separate team's responsibility
* Teams lack security knowledge and skills
* Security practices are inconsistent across teams
* Documentation is missing or outdated
* Security improvements happen in isolation
* Maturity improvements lack structure and measurement

This lab addresses these by:

* Creating enablement programs that build security skills
* Establishing standards that ensure consistency
* Forming communities that share knowledge
* Providing roadmaps that guide improvement
* Coaching teams to adopt security practices
* Measuring maturity to track progress

== Your Mission
Build enablement and maturity that outlast one security hero. Theater does not stop attackers—practices that teams actually run do.

== Understanding the Architecture

[cols="1,2,2",options="header"]
|===
|Component | Purpose | Implementation Approach
|Enablement Programs | Build security skills across teams | Training programs, workshops, hands-on labs, certifications
|Documentation Standards | Ensure consistent security practices | Templates, style guides, best practices, runbooks
|Internal Guilds | Create security communities | Security guilds, communities of practice, regular meetings
|Maturity Roadmaps | Guide security improvement journey | Maturity models, assessment frameworks, improvement plans
|Coaching | Support teams in security adoption | One-on-one coaching, team workshops, mentoring
|Measurement | Track security culture maturity | Surveys, assessments, metrics, KPIs
|===
