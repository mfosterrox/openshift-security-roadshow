== Overview

These labs go deep on Red Hat Advanced Cluster Security (RHACS): visibility, vulnerability and risk management, policy, CI/CD gates, compliance, notifications, API, network security, and installation.

Use this track after (or alongside) the role path that owns the capability—especially DevSecOps, Platform Operator, and Security Architect. Related framework labs include xref:301-04-runtime-threat-detection.adoc[301-04], xref:301-05-incident-response-forensics.adoc[301-05], and xref:301-07-automated-remediation.adoc[301-07].

Audience and prerequisites:

* OpenShift cluster with RHACS Central and secured clusters available (this environment is pre-configured)
* Familiarity with OpenShift console navigation and basic CLI (`oc`, `roxctl`)
* Credentials and console links are on the xref:index.adoc[home page]
